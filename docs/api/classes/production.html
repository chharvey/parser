Class Production

A Production is an item in a formal context-free grammar. It consists of a nonterminal on the left-hand side, which serves as the identifier of the production, and on the right-hand side one ore more choices, or sequences of terminals and/or nonterminals, which can be reduced to the left-hand side nonterminal in a parsing action.
