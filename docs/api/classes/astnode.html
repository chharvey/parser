Class ASTNode

An ASTNode is a node in the Abstract Syntax Tree and holds only the semantics of a ParseNode.

An ASTNode is an abstraction of a ParseNode, without syntactic details. For example, the expression 5 + 2 * 3 can be represented by the following abstract tree:

<ASTNodeOperation operator="+">
    <ASTNodeConstant value="5"/>
    <ASTNodeOperation operator="*">
        <ASTNodeConstant value="2"/>
        <ASTNodeConstant value="3"/>
    </ASTNodeOperation>
</ASTNodeOperation>
